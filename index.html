🔒 Ajuste de Segurança de Linha (RLS) Necessário! Este erro de "violates row-level security policy" ocorre porque o Supabase bloqueia a gravação na tabela tecnicos por padrão por questões de segurança.

Como resolver agora de forma simples e rápida:
1. Acesse o painel do seu projeto no Supabase.
2. No menu lateral esquerdo, clique em SQL Editor (ícone de terminal SQL >_) e clique em "New Query" (Nova Consulta).
3. Cole o código SQL abaixo para desativar a restrição RLS nesta tabela específica e clique no botão "Run" (Executar) no canto inferior direito:
ALTER TABLE tecnicos DISABLE ROW LEVEL SECURITY;
4. Feito isso, volte aqui e tente salvar o técnico novamente! O erro irá sumir imediatamente.
